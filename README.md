# Student---management--c# 📚 Student Management System (C++)

## 📖 Project Description
This is a simple Student Management System built using C++.
It allows adding, displaying, and searching student records using a menu-driven program.

## ⚙️ Features
- Add student details
- Display all students
- Search student by roll number
- Simple menu-based system

## 🛠️ Technologies Used
- C++
- Vector (STL)
- Console Application

## 💻 How to Run
1. Open code in Cxxdroid / CodeBlocks / VS Code
2. Compile the program
3. Run it
4. Use menu options
